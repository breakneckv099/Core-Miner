# Core Miner V1.0.0
A simple cpu/gpu crypto miner pretty easy to use if you know what you are doing.

Heres the virus total on the latest version: [VirusTotal](https://www.virustotal.com/gui/file/252706828bd97ec77b45778b6341856e94818874710d0c1583c2cce6db6e0a30?nocache=1).
A few false positives because its a crypto miner and its virus total but yeah if yall wanna do your own scans go ahead.

Requirements:
- Python 3.8+
- Decent computer
- Stable internet connection
- Stable power supply
- A brain and youtube


How to setup/use:
- Download the project as a .zip and extract the .zip anywhere
- Run the requirements.bat and wait for it to install the dependencies
- Run the Core Miner.exe and wait for it to open
- Enter all of your settings and your wallet then hit Save Config
- Next go to Downloads and select Download for Selected Coin or Download All Miners
- After that hit Check Prereqs and make sure you have everything you need
- Bonus: Before mining if you want to zoom out in the Monitor tab it explains there
- Lastly go back to Mine and hit Start Mining you can check in Logs to see if it started or go and watch in Monitor
